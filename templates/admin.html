API State

  • API State : indicates if the proxy is up or down,
  • API Health : an indication of how well our requests performs influenced by failed requests over successful requests, if the API stays angry for too long it may be an indication that something is wrong either with the proxy or the Property Data API
  • Shutdown API: shutting down the proxy may be helpful in-case of misuse or to trouble shoot traffic issues. either way its a quick emergency shutoff button
Stats indicates all requests going through the API Proxy, and may help you decide how well this proxy is doing, saving you money and time.
